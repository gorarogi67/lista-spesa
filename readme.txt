# Lista della Spesa - Istruzioni

## Come utilizzare l'applicazione offline con salvataggio dati

L'applicazione "Lista della Spesa" è progettata per funzionare offline e salvare i tuoi dati anche quando chiudi il browser. Per utilizzarla correttamente e beneficiare della funzionalità di salvataggio, segui queste istruzioni:

### Installazione

1. **Scarica il file HTML**
   - Salva il file `lista-spesa.html` sul tuo dispositivo (computer, tablet o smartphone)
   - Su PC: Fai clic destro sul link del file e seleziona "Salva link come..."
   - Su smartphone: Tieni premuto sul link e seleziona l'opzione per scaricare o salvare il file

2. **Apri l'applicazione**
   - Sul tuo computer: Fai doppio clic sul file `lista-spesa.html` per aprirlo nel browser predefinito
   - Su iPhone/iPad: 
     - Apri l'app "File"
     - Vai nel percorso dove hai salvato il file
     - Tocca il file per aprirlo in Safari
   - Su Android:
     - Vai nella cartella dei download
     - Tocca il file per aprirlo nel browser predefinito

### Funzionalità principali

- **Aggiungere prodotti**: Digita il nome del prodotto nella barra principale e premi "Aggiungi" o il tasto Invio
- **Aggiungere più prodotti in una volta**: Premi "Inserisci testo con lista spesa" e inserisci un testo contenente tutti i prodotti (separati da virgole, punti o su righe diverse)
- **Rimuovere un prodotto**: Quando metti un prodotto nel carrello durante la spesa, premi il pulsante "Nel carrello" per rimuoverlo dalla lista
- **Cancellare tutta la lista**: Usa il pulsante "Cancella tutta la lista" in fondo alla pagina
- **Esportare la lista**: Premi "Esporta lista" per salvare un file JSON con i tuoi prodotti (utile come backup)
- **Importare la lista**: Premi "Importa lista" per caricare una lista precedentemente esportata

### Salvataggio automatico

Quando apri l'applicazione direttamente sul tuo dispositivo (non attraverso un'interfaccia web come Claude), l'app salverà automaticamente la tua lista della spesa. In questo modo:

- La tua lista verrà memorizzata anche quando chiudi il browser
- Quando riapri l'applicazione, ritroverai tutti i tuoi prodotti
- Ogni modifica viene salvata in tempo reale

Se l'applicazione funziona correttamente con il salvataggio, vedrai un messaggio verde in fondo alla pagina: "✓ I tuoi articoli vengono salvati automaticamente"

### Risoluzione dei problemi

Se vedi il messaggio "⚠️ La tua lista non verrà salvata quando chiudi la pagina":

1. Assicurati di aver aperto il file direttamente dal tuo dispositivo (non tramite un link web)
2. Prova a utilizzare un browser diverso (Chrome, Safari, Firefox)
3. Utilizza la funzione "Esporta lista" per creare backup manuali dei tuoi dati

### Compatibilità

L'applicazione funziona su tutti i dispositivi moderni:
- PC/Mac (Chrome, Firefox, Safari, Edge)
- iPhone/iPad (Safari)
- Android (Chrome, Samsung Internet)

### Utilizzo offline

Una volta scaricato il file, non è necessaria alcuna connessione Internet per utilizzare l'applicazione. Puoi usarla anche quando sei in negozio senza Wi-Fi o connessione dati.
